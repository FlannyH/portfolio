Flan Game Boy Emulator

Made using Rust

Platforms: Windows

Solo Project

2022

Description

This is one of my first Rust projects, where I made a Game Boy emulator to get more familiar with the language.

The emulator can emulate games like Pokemon and Super Mario Land, fully playable at full speed, with stereo audio, a custom bootrom (with the option to provide the official bootrom). It's strictly a Game Boy emulator, and does not support Game Boy Color features.

The code has a flag for a debug mode, which will visualize useful information, like the current state of video memory (on the left), currently loaded ROM banks (in the middle), and palettes (below the ROM banks).
