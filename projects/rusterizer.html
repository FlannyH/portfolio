Rusterizer

Made using Rust

Platforms: Windows

Solo Project

2023

Description

To learn the Rust language better, I followed a mini-course on creating a software rasterizer in Rust.

It has perspective correct texture mapping, with bilinear filtering, multiple wrapping modes, and mip-mapping. It also supports glTF loading with correct scene hierarchy and local transformations applied. And lastly, there is a debug camera to move around the scene.
