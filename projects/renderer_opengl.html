OpenGL renderer

Made using C++ and OpenGL

Plaftorms: Windows

Solo Project

Description

For my personal development, I built a renderer in C++ and OpenGL. It has a PBR shader, glTF loading, a custom memory allocator with debugger.

I did at one point try to implement a DirectX 12, but it does not work yet. When building the project, make sure to select one of the OpenGL profiles.

On the image below this paragraph, on the left you see the memory debugger, which shows every currently allocated memory chunks. You specify a memory label by setting the `curr_memory_chunk_label` string property of the allocator to a description of the memory chunk.

At the bottom, you see the console log output, and on the right is a space for debug tweaks. Currently, you can tweak the roughness power, which changes how the reflections look. You can also flip the normal map's green channel between OpenGL mode and DirectX mode.
