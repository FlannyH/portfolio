Physically Based Renderer

Made using C++ and DirectX 12

Platforms: Windows

Solo Project

2025

Description

Building on what I learned when making my old OpenGL renderer, combined with what I learned at university and at my internship, I started working on a renderer using DirectX 12.

Using DirectX 12 over OpenGL means I have lower level access to the GPU, and allows me to use modern features like hardware accelerated ray tracing (RTX). I chose DirectX 12 over Vulkan, because I learned more about DirectX 12 than Vulkan at this point.

The aim of this project is to build a real-time renderer (60 FPS), which looks as close to the reference pathtracer as I can get. I'm currently running an RTX 3060 12 GB, but it would be very nice if the real-time renderer also runs at acceptable frame rates on low end hardware like the GTX 1050, perhaps using a lower quality setting.

You can switch between the pathtracer renderer and the real-time using the number keys, where 1 is the work-in-progress real-time renderer, and 2 is the reference pathtracer.
